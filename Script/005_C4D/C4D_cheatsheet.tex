\chapter{基本操作}
\begin{itemize}
    \item 更改分辨率：打开渲染器设置，Ctrl + B
    \item 旋转视图：按住Alt + 鼠标
    \item 新建材质：双击材质栏
    \item 新增材料属性：双击材质球
    \item 赋予材料：将材质球拖放至几何体上
    \item 旋转：R键
    \item 缩放：T键
    \item 按住Shift调节，以10为单位
    \item 按住Alt调节，以0.1位单位
    \item 单击右键，复位
    \item 线条着色：N + B
    \item 连接对象并删除
    \item 按住ctrl键进行选择是消除
    \item 按住ctrl键拖拽中键是改变选择范围
    \item 循环选择，选择一圈点，U + L
    \item 按住ctrl进行拖拽
    \item 内部挤压，i键
    \item 挤压，D键
    \item 切刀工具，k键，在平面视图操作
    \item 两个面相接的地方如果点没有相连，那么需要优化
    \item 焊接两个点，M-Q
    \item 断开一条线，先切刀，然后右键选择断开连接
    \item 调用弹出菜单，V
    \item 显示材质颜色，点击shift+V，查看-纹理
    \item 删除多余点，选中-消除
    \item 
\end{itemize}
